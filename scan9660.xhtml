scan9660

Do you have a very old data CD, with extremely valuable information on it that has accidentally been very badly damaged? Due to extreme negligence, do you also have no other backups of this information? In addition, is the very start of the disc damaged, meaning the filesystem can't even be mounted? Is the filesystem ISO9660? Then congratulations, this is the tool for you!

scan9660 uses a heuristic search to look for directory records on ISO9660 images. It then extracts any available information from the disk and loosely recreates the file structure on your local disk. It is best used in combination with ddrescue in order to read as much of the relevant image as possible.

scan9660 is written in SISC Scheme. SISC users can run it with this command line:

sisc -c main main.scm args

Before resorting to using scan9660, make sure you use ddrescue. The image may be mountable through normal means on the loopback device. To be sure you've retrieved as much of the image as possible, repeatedly run ddrescue with the -r 1 option, examining the error count each time. When the error count doesn't change anymore, the maximum amount of data has been retrieved from the CD. See also my script iterative_rescue.sh, which automates this process.
